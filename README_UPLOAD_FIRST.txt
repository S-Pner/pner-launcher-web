# PnerLauncher GitHub Pages Starter

이 폴더 안의 파일들을 GitHub 저장소 `pner-launcher-web`에 업로드하세요.

필수 파일:
- index.html
- servers.json
- launcher-config.json
- assets/ 폴더

GitHub Pages가 켜진 뒤 아래 주소가 열리면 성공입니다.

https://sin1388211.github.io/pner-launcher-web/servers.json

주의:
- 서버 IP, 포트, 마크 버전은 servers.json에서 수정합니다.
- GitHub Pages에는 비밀번호, 토큰, 개인정보를 올리지 마세요.
